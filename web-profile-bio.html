Tentang saya..

Saya lahir di kota Malang pada tanggal 20 Februari 1996 dan bekerja sebagai admin pengiriman di sebuah toko online yang menjual peralatan ikan. Namun, lebih dari itu, saya tertarik untuk mempelajari programming dan fokus dalam mengembangkan keahlian saya dalam bidang ini. Saat ini, saya mengambil kursus online diwaktu luang saya untuk belajar coding dan memperdalam pemahaman saya tentang bahasa pemrograman. Saya menemukan bahwa saya sangat menikmati belajar tentang teknologi dan terus mencari tahu cara menggunakan teknologi untuk menciptakan sesuatu yang bermanfaat bagi orang lain. Saya merasa senang ketika bisa menyelesaikan sebuah proyek dan melihat hasilnya. Selain itu, saya juga masih tetap mengikuti hobi saya, yaitu touring motor dan bermain game role-playing, meskipun kini lebih banyak waktu saya habiskan untuk bekerja. Saya yakin bahwa dengan semangat dan kerja keras, saya dapat mencapai tujuan saya dan menjadi ahli dalam bidang programming.
